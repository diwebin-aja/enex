Cara Menemukan User ID

Untuk melakukan top-up, kamu memerlukan User ID dan Zone ID dari akun Mobile Legends kamu. Ikuti langkah berikut:

1
Buka aplikasi Mobile Legends dan masuk ke layar utama game (Lobby).
2
Ketuk Avatar / Foto Profil kamu yang berada di pojok kiri atas layar.
3
Pada halaman profil, kamu akan melihat deretan angka di bawah nama akunmu. Contoh: 12345678 (1234). Angka di luar kurung adalah User ID, dan angka di dalam kurung adalah Zone ID.

Cara Membeli Diamond

Setelah kamu mencatat User ID dan Zone ID, ikuti langkah mudah ini untuk menyelesaikan pembelian:

1
Pilih nominal Diamond, Weekly Pass, atau Membership pada daftar produk di atas.
2
Gulir ke bawah, lalu masukkan User ID dan Zone ID kamu ke dalam kolom form yang tersedia.
3
Klik tombol Pilih metode pembayaran, selesaikan transaksi, dan Diamond akan langsung masuk ke akunmu dalam hitungan detik!
